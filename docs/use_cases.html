Actor Capabilities

👤

Regular User

Donors and Recipients

Blood Request/Donation
Initiate urgent blood requests or schedule voluntary donations at specific hospitals.
Nearby Search
Use geolocation to find the closest hospitals with available blood types.
Communication
Chat directly with hospital admins regarding active requests and receive status alerts.
🏥

Hospital Admin

Facility Managers

Request Processing
Review incoming donor and recipient applications; approve, reject, or mark as 'on progress'.
Inventory Control
Update and sync blood unit stock levels for visibility to the public.
Active Response
Manage real-time chats with users and send custom feedback messages.
🛡️

Super Admin

System Governors

Security & Governance
Ban malicious users and monitor global transaction logs for system integrity.
Infrastructure Management
Onboard new hospitals, manage directories, and assign admin permissions.
Ecosystem Oversight
View global analytics on blood availability and request fulfillment rates.
